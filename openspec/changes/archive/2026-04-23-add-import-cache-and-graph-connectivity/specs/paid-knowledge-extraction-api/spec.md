## ADDED Requirements

### Requirement: 支持同链接导入失败时的缓存回退
系统 MUST 在白名单新闻链接首次成功导入后缓存结构化结果，并在后续同链接实时抓取失败时优先回退到本地缓存。

#### Scenario: 成功导入后写入缓存
- **当** 系统成功导入某个白名单新闻链接
- **则** 系统写入该链接对应的本地缓存
- **且** 缓存至少包含标题、正文、来源站点和缓存时间

#### Scenario: redirect 或 canonical URL 命中同一缓存
- **当** 某个白名单新闻链接首次成功导入后发生站内 redirect 或 canonical URL 归一
- **则** 系统后续使用规范化后的请求 URL 或 finalUrl 都可以命中同一份缓存
- **且** 不会因为 URL 细节差异重复创建多份等价缓存

#### Scenario: 同链接抓取失败时回退缓存
- **当** 某个曾成功导入过的白名单新闻链接再次导入且实时抓取失败
- **则** 系统返回该链接最近一次成功缓存的导入结果
- **且** 返回结果明确标记当前使用了缓存回退

#### Scenario: 无缓存时仍然明确失败
- **当** 白名单新闻链接导入失败且本地不存在该链接缓存
- **则** 系统返回明确的导入失败信息
- **且** 不伪造成功导入结果

### Requirement: 图谱展示优先保持连通性
系统 MUST 对知识图谱展示层做连通性后处理，尽量避免多个孤立节点直接呈现在页面上。

#### Scenario: 已有关系边时隐藏孤立节点
- **当** 某次 session 的原始图谱中已存在至少一条关系边
- **则** 系统优先展示有边连接的节点
- **且** 默认不展示零度孤立节点

#### Scenario: 完全无边时补充轻量辅助边
- **当** 某次 session 的原始图谱没有任何关系边但存在多个节点
- **则** 系统在展示层自动补充 `1-3` 条轻量辅助边
- **且** 原始 `relations` 数据保持不变
- **且** 辅助边带有可区分的来源标记

## MODIFIED Requirements

### Requirement: 图谱页面展示导入来源元数据
系统 MUST 在链接导入成功后，将来源链接、来源站点和导入标题贯穿到可审阅的图谱展示层。

#### Scenario: graph 页面展示来源信息
- **当** 某次 session 由白名单文章链接导入而来
- **则** 对应 graph 页面展示原始链接、来源站点和导入标题
- **且** live result 与 agent session 产物中的来源元数据保持一致

#### Scenario: graph 页面展示缓存回退状态
- **当** 某次链接导入因实时抓取失败而回退到本地缓存
- **则** graph 页面展示该 session 当前使用了缓存回退
- **且** 页面可展示缓存时间等辅助信息

#### Scenario: graph 页面展示实时抓取状态
- **当** 某次链接导入使用实时抓取成功完成
- **则** graph 页面展示当前为实时抓取结果
- **且** 不会错误标记为缓存回退

#### Scenario: graph 页面说明辅助边仅用于展示
- **当** 图谱展示层包含辅助连接边
- **则** graph 页面有可见文案说明该类边仅用于展示连通性
- **且** 不与原始抽取关系混淆

#### Scenario: live session payload 展示来源信息
- **当** 页面读取 `latest`、指定 `sessionId` 或轮询某次由链接导入产生的 live session
- **则** 返回的 live session payload 包含 `articleUrl`、`sourceSite` 和导入方式等来源元数据
- **且** 这些元数据与对应的 agent session 产物保持一致

#### Scenario: live session payload 展示缓存回退状态
- **当** 某次链接导入使用本地缓存回退
- **则** 返回的 live session payload 明确标记 `importStatus='cache'`
- **且** 如有缓存时间则一并返回 `cachedAt`
- **且** 页面完成态可据此展示缓存回退状态

#### Scenario: live session payload 展示实时抓取状态
- **当** 某次链接导入使用实时抓取成功完成
- **则** 返回的 live session payload 明确标记 `importStatus='live'`
- **且** 页面完成态可据此展示实时抓取状态
